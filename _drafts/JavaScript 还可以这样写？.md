 ### 求幂运算符「**」

```js
2 ** 10 // 1024
2 ** -1 // 0.5
```

当数字前出现一元运算符`+/-/~/!/delete/void/typeof`时，需要明确指定运算顺序，否则会提示语法错误。

```
-2 ** 2 // SyntaxError
(-2) ** 2 // 4
-(2 ** 2) // -4
```

要注意它是从右向左结合的。

```js
2 ** 2 ** 3 // 256
2 ** (2 ** 3) // 256
(2 ** 2) ** 3 // 64
```


### 移位运算符取整「>> <<」

```js
3.14 >> 0 // 3
3 / 2 >> 1 // 1
1024 / 12 << 0 // 44
```
