declare [+/-][选项] 变量名
	- 给变量设定类型属性
	+ 取消变量的类型属性
	-a 将变量声明为数组类型
	-i 将变量声明为整数型
	-x 将变量声明为环境变量
	-r 将变量声明为只读变量，只读属性让变量不能修改、删除、取消类型，临时环境下重启才会消失
	-p 查询所有变量的属性，显示指定变量的类型

声明为整型，可进行数值运算
```
a=1
b=2
declare -i c=$a+$b  
```
export 利用 declare 来声明环境变量

$((运算式))   $[运算式]
```
a=1
b=2
c=$(( $a+$b ))
d=$[ $a+$b ]
```
