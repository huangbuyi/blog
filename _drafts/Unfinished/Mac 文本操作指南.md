        
# 跳转光标
Command + ←                 行首
Command + →                 行尾
Command + ↑                 文首
Command + ↓                 文尾
Ctrl + M                    跳转到对应括号
Alt + ←                     向左跳过一个单词
Alt + →                     向右跳过一个单词

## 搜索栏
（「#」 搜索变量名；「:」跳转第几行；「@」 搜索函数名）
Command + P                 搜索项目中的文件
Command + R                 搜索函数名
Ctrl + G                    跳转到第几行
Command + F                 查找内容
Command + Shift + F         查找替换

## 视图
Command + Alt + Num         分屏
Ctrl + Num                  切换分屏
Command + K + B             隐藏侧边栏
Command + W                 关闭当前文件
Command + Shift + W         关闭所有打开文件
Command + N                 新建
Command + Num               切换窗口

## 选中
Command + D                 选中下一个同名元素
option + drap mouse         垂直选中
Ctrl + Shift + ← or →       选中词
Cammand + Shift + Space     拓展选区，连续按继续拓展
Command + Shift + M         选中括号内的内容
Command + Ctrl + G          选中所有同名元素
Command + L                 选中行

## 文本操作
Command + X                 删除当前 
Command + /                 注释
Tab                         缩进
Shift + Tab                 左缩进
Command + K + U             转为大写
Command + K + L             转为小写
Command + Shift + /         多行注释
Command + Ctrl + ↑ 或 ↓     下或下移当前行
Ctrl + Shift + Enter        在上一行插入空行
Ctrl + Enter                在下一行插入空行，无论光标位置
Ctrl + J                    合并
Ctrl + Shift + D            复制行
Ctrl + K                    删至行尾
Ctrl + Shift + K            删除行 

## 命令栏
Command + Shift + P         打开命令栏
