## 变量命名规则

变量名必须字母或下划线打头，由字母、数字和下划线组成
长度不超过 255 个字符
变量名有效范围内必须是唯一的
Bash 默认类型是字符串型

## 变量分类
用户自定义变量
环境变量：保存和系统操作环境相关的数据
位置参数变量：用来向脚本当中传递参数或数据，变量名不能自定义，变量作用是固定的
预定义变量：Bash 已经定义好的变量，变量名不能自定义，变量作用是固定的

## 定义变量
变量名=变量值 （无空格）
变量名= “变量值” （允许等号后有空格）
变量名= ‘变量值’ （允许等号后有空格）
双引号内特殊值会被转义，单引号不会

变量叠加：
"$变量"新值 
${变量}新值 

## set
set 查询所有正在运行的变量
    -u 变量不存在时报错（默认为空）

unset 变量 （删除变量，不需要加「$」符）

## Bash 环境变量
bash 进入子 bash
pstree 查看进程树，可以查看 bash 的层级
环境变量是全局的，在所有 shell 中共享，用户定义变量只在 当前 shell 可见

设置环境变量：
export 变量名
export 变量名=变量值

env （查看环境变量）
环境变量建议大写

### PATH
$PATH 系统搜索命令路径

脚本开头：#!/bin/bash
chmod 755 hello.sh 权限
将脚本放到 $PATH 路径中，或把脚本所在路径添加到 $PATH 中，都可以使脚本在全局中执行
$PATH 临时生效

### PS1
PS1 变量：命令提示符设置
\d 显示日期 
\H 完整主机名 
\t 24 小时制时间（HH:MM:SS）
\A 24 小时制时间（HH:MM）
\u 显示当前用户名
\w 显示当前目录完整名称
\W 显示当前目录最后一个目录名称
\$ 提示符，root 用户显示为「#」，普通用户显示为「$」
例如：\h:\W \u\$ 显示为： HuangdeMBP:~ huangshibin$
（临时生效）

### 位置参数变量
位置参数变量 作用
$n         n 为数字，$0 代表命令本身，$1-$9 代表第一到第九个参数，十以上需要大括号包含，如 ${10}
$*         代表命令行中所有参数，所有参数为一个整体
$@         代表命令行中所有参数，每个参数区别对待
$#         所有参数的个数
```
#!/bin/bash
num1=$1
num2=$2
sum=$(($num1 + $num2)) # 不适用括号进行字符串运算，加括号为数值运算
echo $sum
# 将第一个参数和第二个参数相加的和，打印出来
```

```
#!/bin/bash
# 只会循环一次
for i in "$*"
    do
        echo "The parameters is: $i"
    done

# 有几个参数循环几次
for j in "$@"
    do 
        echo "Parameter: $j"
    done
```

### 预定义变量
预定义变量 作用
$?        最后一次执行命令的返回状态，值为 0，上一次执行正确执行，值为非 0，上一次执行不正确
$$        当前进程的进程号（PID）
$!        后台运行的最后一个进程的进程号（PID）
$!

### read
read [选项] [变量名]
-p “提示信息”：等待 read 输入时，输出提示信息
-t 指定等待时间
-n 命令接受指定字符数就会执行
-s 隐藏输入数据，用于机密信息输入
```
#!/bin/bash
read -p "Please input your name: " name
echo $name

read -p "Please input your passwd: " -s passwd
echo -e "\n"
echo $passwd

read -p "Please input your gender [M/F]: " -n 1 sex  # 控制为一个字符
echo -e
echo $sex
```
