Ctrl + Cmd + f          全屏切换

## 切换
Cmd + `                 切换下一个窗口
Cmd + Shift + `         切换上一个窗口
Cmd + Tab               切换下一个程序
Cmd + Shift + Tab       切换上一个程序
Ctl + Tab               切换下一个标签
Ctl + Shift + Tab       切换上一个标签
Ctl + ← 或 Ctl + →       切换全屏窗口


## 关闭
Cmd + W                 关闭标签或窗口
Cmd + Q                 退出程序
Cmd + H                 隐藏窗口
Cmd + Alt + H           隐藏其它窗口
